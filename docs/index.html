Swift-Kuery-PostgreSQL

PostgreSQL plugin for Swift-Kuery framework

Summary

PostgreSQL plugin for Swift-Kuery framework.

PostgreSQL installation

Linux

$ apt-get install libpq-dev

macOS

$ brew install postgresql

Running Swift-Kuery-PostgreSQL

First create an instance of Swift-Kuery-PostgreSQL by calling:

public required init(host: String, port: Int32, options: [ConnectionOptions]?)

Where: - host and port are the host and the port of PostgreSQL - ConnectionOptions an optional set of: * options - command-line options to be sent to the server * databaseName - the database name * userName - the user name * password - the user password * connectionTimeout - maximum wait for connection in seconds. Zero or not specified means wait indefinitely.

For more information please see PostgreSQL manual.

To establish a connection call:

public func connect(onCompletion: (QueryError?) -> ())
